## One-Liner

Open-sources attribution graph tools that reveal internal model reasoning steps, enabling anyone to trace circuits in open-weights models.

## Key Points

- Attribution graphs partially reveal the internal steps a model took to produce a particular output
- Open-source library supports generating graphs on popular open-weights models
- Neuronpedia frontend enables interactive exploration, annotation, and sharing of graphs
- Researchers can test hypotheses by modifying feature values and observing output changes
- Built on "Tracing the thoughts of a large language model" research
